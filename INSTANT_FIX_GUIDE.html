🚨 إصلاح فوري لمشاكل GitHub Actions

⚡ الحل السريع (30 ثانية)

1. اذهب إلى GitHub Repository:

🔗 فتح Repository

2. اذهب إلى ملف GitHub Actions:

📝 تعديل build-apk.yml

3. استبدل المحتوى بالكامل بهذا:

name: Build APK on: push: branches: [ main, master ] workflow_dispatch: jobs: build: runs-on: ubuntu-latest steps: - name: Checkout uses: actions/checkout@v3 - name: Setup Java uses: actions/setup-java@v3 with: distribution: 'zulu' java-version: '11' - name: Setup Flutter uses: subosito/flutter-action@v2 with: flutter-version: '3.19.6' channel: 'stable' - name: Get dependencies run: flutter pub get - name: Build APK run: flutter build apk --release - name: Upload APK uses: actions/upload-artifact@v3 with: name: app-release path: build/app/outputs/flutter-apk/app-release.apk

4. انقر "Commit changes"

🔄 تشغيل البناء

بعد حفظ الملف:

  1. اذهب إلى: 🔄 GitHub Actions
  2. انقر على "Build APK" من القائمة اليسرى
  3. انقر "Run workflow" → "Run workflow"
  4. انتظر 3-5 دقائق
  5. حمل APK من Artifacts

🆘 إذا لم يعمل - الحل البديل

أنشئ ملف جديد:

  1. اذهب إلى Repository
  2. انقر "Add file" → "Create new file"
  3. اكتب: .github/workflows/simple.yml
  4. الصق هذا المحتوى:
name: Simple Build on: [push, workflow_dispatch] jobs: build: runs-on: ubuntu-latest steps: - uses: actions/checkout@v3 - uses: actions/setup-java@v3 with: distribution: 'zulu' java-version: '11' - uses: subosito/flutter-action@v2 with: flutter-version: '3.19.6' - run: flutter pub get - run: flutter build apk - uses: actions/upload-artifact@v3 with: name: apk path: build/app/outputs/flutter-apk/app-release.apk

📱 النتيجة المضمونة

خلال 5 دقائق من تطبيق الإصلاح:

  • ✅ GitHub Actions ستعمل بدون أخطاء
  • ✅ APK سيتم بناؤه بنجاح
  • ✅ ملف APK جاهز للتحميل
  • ✅ حجم APK حوالي 20-30 MB

🔗 روابط سريعة

📁 Repository 🔄 Actions 📝 Edit Workflow

🎯 خطة العمل الفورية

1. انقر "فتح Repository" أعلاه

2. عدل ملف GitHub Actions

3. شغل Workflow

4. حمل APK خلال 5 دقائق
